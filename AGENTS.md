# AGENTS.md（中文翻译）

## 项目概述

本仓库是一个结构化的知识项目，主题为 **AI 所需的数学基础**。

目标是生成一套高质量的 Markdown 文档，系统性地讲解 AI 所需的数学基础，包括：

* 线性代数
* 微积分
* 概率论与数理统计
* 优化理论
* 离散数学与图论
* 按方向扩展的专题（如信息论、几何、强化学习相关数学等）

最终内容应满足：

* 适合自学
* 适合作为知识库构建
* 适合未来扩展为教程 / 笔记 / 课程材料
* 使用 **中文** 编写
* 数学上严谨，但不过度形式化
* 面向 **AI / 机器学习 / 深度学习** 应用场景

---

## Agent 的核心任务

在本仓库中工作时，你的任务 **不是编写泛泛的数学笔记**。

你的任务是创建和维护 Markdown 文档，用来回答：

> “AI 需要哪些数学知识？为什么需要？应该学到什么程度？这些知识如何与实际 AI 方法建立联系？”

每一份文档都应帮助读者建立以下四个方面的联系：

1. **数学概念本身**
2. **它在 AI 中的作用**
3. **学习时应达到的理解深度**
4. **在典型算法或模型中的具体体现**

---

## 目标读者

假设读者属于以下人群之一：

* 想系统学习 AI 数学基础的初学者
* 已会使用框架，但数学基础不够扎实的工程实践者
* 希望为阅读机器学习 / 深度学习 / 大模型论文做准备的读者

默认前提：

* 具备高中数学基础
* 对抽象证明不太熟悉
* 需要直觉 + 形式化 + AI 应用背景的结合

因此应做到：

* 清晰解释概念
* 不跳过必要的直觉
* 非必要不过度展开严格证明
* 强调数学在 AI 中的实际用途

---

## 语言与写作规范

### 语言要求

* 所有内容必须使用 **简体中文** 编写
* 重要术语首次出现时需附带英文，例如：

  * 线性代数（Linear Algebra）
  * 梯度（Gradient）
  * 贝叶斯定理（Bayes' Theorem）

### 写作风格

应具备以下特点：

* 清晰
* 结构化
* 教学导向
* 简洁但不枯燥
* 类似高质量技术学习笔记，而非博客文章

避免：

* 过于随意的网络语言
* 夸张营销式表达
* 空洞的励志句
* 未解释的符号堆砌

### 排版建议

优先采用以下结构：

* 清晰的标题层级
* 简短段落说明
* 必要但克制的列表
* 使用 LaTeX 表示公式
* 在合适位置加入示例
* 仅在确有必要时使用表格
