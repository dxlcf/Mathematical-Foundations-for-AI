# Mathematical-Foundations-for-AI (AI 需要的数学基础)

这是一个结构化的知识项目，旨在系统性地讲解人工智能（AI）领域所需的数学基础理论。

## 🎯 项目目标

本项目的目标是梳理并沉淀一套高质量的 Markdown 文档。这些文档**不是泛泛的数学笔记**，而是着重帮助读者建立以下四个方面的联系：

1. **数学概念本身**
2. 它在 **AI 中的核心作用**
3. 学习时应达到的**理解深度**
4. 在**典型算法或模型中**的具体体现

一言以蔽之：
> **用线性代数表示模型，用微积分训练模型，用概率统计理解数据，用优化方法把模型学出来。**

## 👥 目标读者

本项目假设读者具备**高中数学基础**，对抽象定理证明不太熟悉，因此内容设计兼顾直觉理解与形式化推导的结合。

如果你属于以下人群，这个项目非常适合你：
* 想系统学习 AI 数学基础的初学者。
* 已经能够调用机器学习框架，但深感原理掌握不扎实的工程实践者。
* 希望能够独立无障碍阅读前沿机器学习、深度学习及大模型（LLMs）论文的进阶读者。

## 📚 知识体系概览

针对实际的 AI 应用环境，我们将数学划分为这几个最重要的板块：

### 一、 线性代数 —— AI 的"通用语言"
AI 中几乎所有数据（图像像素、文本词嵌入、模型权重）最终都会表示为向量、矩阵或张量。神经网络每一层的核心操作，本质上就是线性变换（加平移组成的仿射变换）。
* **必背核心：** 矩阵运算、特征值/特征向量(PCA)、奇异值分解(SVD)。

### 二、 微积分 —— 让模型"学习"的引擎
参数应该往哪个方向调整才能使模型误差减小？这就完全依赖微积分。深度学习赖以生存的**反向传播**，即链式法则的大规模应用。
* **必背核心：** 偏导数、梯度/方向导数、链式法则、雅可比与海森矩阵（高阶优化）。

### 三、 概率论与统计 —— 量化不确定性
真实数据往往携带噪声；很多模型的设计本身就是用来估计概率（如Softmax分类，或生成式模型 VAE、扩散模型等）。
* **必背核心：** 贝叶斯定理、常见分布类型、期望/方差、最大似然估计（MLE）。

### 四、 优化理论 —— 找到最优参数
明确了前进方向后，怎样快速且不陷入极值点地下山？神经网络训练本质上是一个极高维的非凸优化问题。
* **必背核心：** 梯度下降（SGD、Adam）、凸优化/非凸优化、鞍点与正则化。

### 五、 离散数学与图论 —— 按方向扩展补充
针对非欧几里得数据结构（如知识图谱、社交网络、分子结构）。
* **必背核心：** 图的矩阵表示（邻接矩阵、拉普拉斯矩阵）、基本算法与复杂度。

## 🗺️ 学习路径与方向侧重

如果你只是想找个最小口径**快速入门做项目**，掌握以下内容即可跨越绝大多数门槛：
👉 **`向量/矩阵乘法 + 导数/链式法则/梯度 + 条件概率/期望/MLE + 梯度下降/学习率`**

### 按阶段学习路径
1. **初级入门：** 线性代数基础 + 微积分基础
2. **中阶突破：** 概率统计 + 多元微分 + 优化入门
3. **高级强化：** 信息论（熵、KL散度） + 离散数学（按需）与专题拓展

### 各个领域的学科偏好
| AI 细分方向 | 核心数学依仗 |
|---|---|
| **深度学习** | 线性代数 + 微积分（链式法则） |
| **机器学习** | 概率统计 + 优化理论 |
| **计算机视觉 (CV)**| 线性代数 + 几何 + 概率 |
| **自然语言处理 (NLP / LLMs)** | 线性代数 + 信息论基础 |
| **强化学习 (RL)** | 概率论 + 动态规划 + 马尔可夫决策过程 (MDP) |
| **图神经网络 (GNN)** | 图论 + 线性代数 |
