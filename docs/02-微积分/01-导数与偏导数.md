# 导数与偏导数

## 1. 导数：变化率的精确描述

### 1.1 直觉

想象你驾车行驶，速度表显示的是你当前的速度——也就是位置关于时间的**瞬时变化率**。这就是导数的本质：描述一个量相对于另一个量如何变化。

在 AI 中，我们关心的是：**损失（Loss）关于模型参数的变化率**。如果某个参数微小增大，损失会增大还是减小？增减多少？导数精确回答这个问题。

### 1.2 形式定义

函数 $f(x)$ 在点 $x_0$ 处的导数（Derivative）定义为：

$$
f'(x_0) = \lim_{h \to 0} \frac{f(x_0 + h) - f(x_0)}{h}
$$

几何意义：这是函数曲线在 $x_0$ 处切线的斜率。

### 1.3 常用求导规则

以下规则在 AI 推导中反复出现，值得熟记：

**幂函数**

$$\frac{d}{dx} x^n = n x^{n-1}$$

**指数函数**（$e$ 为自然常数）

$$\frac{d}{dx} e^x = e^x$$

**对数函数**

$$\frac{d}{dx} \ln x = \frac{1}{x}$$

**乘积法则（Product Rule）**

$$\frac{d}{dx}[f(x) \cdot g(x)] = f'(x) g(x) + f(x) g'(x)$$

**链式法则（Chain Rule）**——将在下一章详细展开

$$\frac{d}{dx} f(g(x)) = f'(g(x)) \cdot g'(x)$$

### 1.4 AI 中的常见激活函数求导

**Sigmoid 函数**：$\sigma(x) = \dfrac{1}{1 + e^{-x}}$

其导数有一个优雅的性质：

$$\sigma'(x) = \sigma(x)(1 - \sigma(x))$$

这使得反向传播中计算 Sigmoid 的梯度非常高效。

**ReLU 函数**：$\text{ReLU}(x) = \max(0, x)$

$$\text{ReLU}'(x) = \begin{cases} 1 & x > 0 \\ 0 & x < 0 \end{cases}$$

在 $x = 0$ 处不可导，实际使用中通常约定为 0 或 1。

## 2. 偏导数：多变量函数的导数

### 2.1 为什么需要偏导数

神经网络的参数数量动辄数百万。损失函数 $L$ 是所有参数的函数：

$$L = L(w_1, w_2, \ldots, w_n)$$

我们需要知道：**每一个参数单独变化时，对损失的影响是多少**。这就是偏导数（Partial Derivative）解决的问题。

### 2.2 定义

函数 $f(x, y)$ 关于 $x$ 的偏导数定义为：固定 $y$ 不变，对 $x$ 求普通导数。

$$\frac{\partial f}{\partial x} = \lim_{h \to 0} \frac{f(x+h, y) - f(x, y)}{h}$$

符号 $\partial$（读作"偏"或"roundd"）专门用于偏导数，区别于普通导数符号 $d$。

### 2.3 示例

设 $f(x, y) = x^2 y + 3xy^2$，则：

$$\frac{\partial f}{\partial x} = 2xy + 3y^2$$

$$\frac{\partial f}{\partial y} = x^2 + 6xy$$

计算方法：对 $x$ 求偏导时，把 $y$ 当作常数；对 $y$ 求偏导时，把 $x$ 当作常数。

### 2.4 在损失函数中的应用

设简单线性模型的损失为：

$$L(w, b) = \frac{1}{n} \sum_{i=1}^{n} (wx_i + b - y_i)^2$$

其偏导数为：

$$\frac{\partial L}{\partial w} = \frac{2}{n} \sum_{i=1}^{n} (wx_i + b - y_i) \cdot x_i$$

$$\frac{\partial L}{\partial b} = \frac{2}{n} \sum_{i=1}^{n} (wx_i + b - y_i)$$

这两个偏导数告诉我们：分别调整权重 $w$ 和偏置 $b$，各自对损失有多大影响——这正是梯度下降的出发点。

## 3. 方向导数：任意方向的变化率

### 3.1 直觉

偏导数只考虑沿坐标轴方向的变化。但实际上，参数空间中可以沿任意方向移动。**方向导数（Directional Derivative）** 描述函数沿某个给定方向的变化率。

### 3.2 定义

设 $\mathbf{u}$ 是一个单位向量，函数 $f$ 沿 $\mathbf{u}$ 方向的方向导数为：

$$D_{\mathbf{u}} f = \nabla f \cdot \mathbf{u}$$

其中 $\nabla f$ 是 $f$ 的梯度向量（下一章详细介绍）。

这个点积的值在 $\mathbf{u}$ 与 $\nabla f$ 方向相同时取最大值——即**梯度方向是函数上升最快的方向**。

## 4. 高阶导数

### 4.1 二阶导数

函数 $f(x)$ 的二阶导数是对导数再次求导：

$$f''(x) = \frac{d^2 f}{dx^2}$$

**几何意义**：描述函数的弯曲程度（曲率）。

- $f''(x) > 0$：函数局部是"碗形"（凸函数），极小值点
- $f''(x) < 0$：函数局部是"山峰形"（凹函数），极大值点
- $f''(x) = 0$：可能是拐点

### 4.2 在优化中的意义

对于损失函数，二阶导数告诉我们：当前所处位置的"地形弯曲度"。这在牛顿法（Newton's Method）等高阶优化方法中被直接利用。对于多变量情形，二阶导数的推广就是**海森矩阵（Hessian Matrix）**，将在第 03 节介绍。

## 小结

| 概念 | 符号 | AI 中的意义 |
|------|------|-------------|
| 导数 | $f'(x)$ 或 $\dfrac{df}{dx}$ | 单参数对损失的影响 |
| 偏导数 | $\dfrac{\partial f}{\partial x_i}$ | 多参数中某一参数对损失的影响 |
| 方向导数 | $D_{\mathbf{u}} f$ | 沿特定方向的变化率 |
| 二阶导数 | $f''(x)$ | 函数曲率，优化的地形信息 |

掌握偏导数的计算是进入反向传播推导的直接前提。建议在继续阅读之前，手动推导几个简单损失函数的偏导数。
