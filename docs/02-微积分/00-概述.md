# 微积分：AI 视角下的概述

## 为什么 AI 需要微积分？

如果说线性代数描述的是"结构"，微积分描述的就是"变化"。

神经网络的训练本质上是一个优化问题：我们希望找到一组参数，使得模型的输出尽可能接近目标。而"如何调整参数"这个问题，正是微积分负责回答的。

没有微积分，就没有反向传播（Backpropagation），也就没有今天几乎所有的深度学习。

## 你需要建立的核心直觉

**导数 = 变化率**

对函数 $f(x)$，导数 $f'(x)$ 告诉你：当 $x$ 微小变化时，$f$ 变化多少。在 AI 中，这意味着：当某个参数微小变化时，损失函数（Loss）变化多少。

**梯度 = 多维导数**

当函数有多个输入变量时，每个方向都有自己的"变化率"。梯度（Gradient）把所有方向的导数打包成一个向量，指向损失**增长最快**的方向。梯度的反方向，就是我们在训练中要走的方向。

**链式法则 = 复合函数的求导规则**

神经网络是层层嵌套的复合函数。链式法则（Chain Rule）让我们能够从输出层一路往回计算每层参数的梯度——这正是反向传播的本质。

## 本章需要达到的理解深度

对于 AI 从业者和学习者，微积分不需要达到数学分析的严格程度，但需要：

- 能够手动计算简单函数的导数和梯度
- 理解链式法则并能追踪计算图中的梯度流动
- 能读懂论文中出现的偏导数符号和梯度表达式
- 对海森矩阵有直觉性理解，知道它在优化中的作用

不需要：

- $\varepsilon$-$\delta$ 极限的严格证明
- 多元积分和向量场的完整理论
- 实变函数或泛函分析
