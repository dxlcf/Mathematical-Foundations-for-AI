# 贝叶斯推断与最大似然（Bayesian Inference & Maximum Likelihood Estimation）

## 1. 核心问题：参数从哪里来？

机器学习模型有参数（权重、偏置等），这些参数需要从数据中**估计**。

估计参数有两种主要的哲学框架：

| 框架 | 核心思想 | 典型方法 |
|------|----------|----------|
| **频率派（Frequentist）** | 参数是固定未知值，用数据估计它 | 最大似然估计（MLE） |
| **贝叶斯派（Bayesian）** | 参数本身是随机变量，用数据更新对它的信念 | 贝叶斯推断、MAP |

两种框架在实践中常常融合使用，深刻理解它们能帮助你更好地理解正则化、损失函数设计和不确定性量化。

---

## 2. 似然函数（Likelihood Function）

### 定义

给定参数 $\theta$，观测数据 $\mathcal{D} = \{x_1, x_2, \ldots, x_n\}$ 的概率（密度）称为**似然函数**：

$$L(\theta) = P(\mathcal{D} \mid \theta) = \prod_{i=1}^n P(x_i \mid \theta)$$

右侧连乘成立的前提是假设各样本**独立同分布**（i.i.d.，independent and identically distributed）。

**重要区分：** 似然函数 $L(\theta)$ 是关于参数 $\theta$ 的函数，数据 $\mathcal{D}$ 是已观测的固定值；概率 $P(\mathcal{D}\mid\theta)$ 是关于数据的函数，$\theta$ 是固定的。形式相同，视角不同。

### 对数似然（Log-Likelihood）

连乘在数值上容易下溢，且求导不方便。实践中几乎总是使用**对数似然**：

$$\ell(\theta) = \log L(\theta) = \sum_{i=1}^n \log P(x_i \mid \theta)$$

由于 $\log$ 是单调递增函数，最大化 $\ell(\theta)$ 等价于最大化 $L(\theta)$。

---

## 3. 最大似然估计（Maximum Likelihood Estimation，MLE）

### 思想

选择使观测数据"最有可能出现"的参数值：

$$\hat{\theta}_{\text{MLE}} = \arg\max_\theta \log P(\mathcal{D} \mid \theta) = \arg\max_\theta \sum_{i=1}^n \log P(x_i \mid \theta)$$

### 示例：高斯分布的 MLE

数据 $\{x_i\}$ 假设服从 $\mathcal{N}(\mu, \sigma^2)$，对数似然为：

$$\ell(\mu, \sigma^2) = -\frac{n}{2}\log(2\pi\sigma^2) - \frac{1}{2\sigma^2}\sum_{i=1}^n (x_i - \mu)^2$$

对 $\mu$ 求导令其为零，得到：

$$\hat{\mu}_{\text{MLE}} = \frac{1}{n}\sum_{i=1}^n x_i$$

即**样本均值**。类似地可得 $\hat{\sigma}^2_{\text{MLE}} = \frac{1}{n}\sum_i (x_i - \hat{\mu})^2$（样本方差）。

这说明：**样本均值和样本方差，正是高斯假设下的 MLE 解**。

### MLE 与交叉熵损失的关系

以分类任务为例，假设输出服从类别分布，真实标签为 $y_i \in \{1,\ldots,K\}$，模型预测概率为 $\hat{p}_k^{(i)}$。

对数似然为：

$$\ell = \sum_{i=1}^n \log \hat{p}_{y_i}^{(i)} = -\sum_{i=1}^n \left(-\log \hat{p}_{y_i}^{(i)}\right)$$

最大化对数似然 $\equiv$ 最小化**交叉熵损失（Cross-Entropy Loss）**：

$$\mathcal{L}_{\text{CE}} = -\frac{1}{n}\sum_{i=1}^n \log \hat{p}_{y_i}^{(i)}$$

**结论：** 分类任务的交叉熵损失，是类别分布假设下 MLE 的直接结果，并非人为设计。

类似地，回归任务中均方误差（MSE）是高斯噪声假设下 MLE 的结果：

$$\text{MSE} = \frac{1}{n}\sum_i (y_i - \hat{y}_i)^2 \quad \Leftarrow \quad \epsilon \sim \mathcal{N}(0, \sigma^2)$$

## 4. 贝叶斯定理回顾

在参数估计场景中，贝叶斯定理写为：

$$\underbrace{P(\theta \mid \mathcal{D})}_{\text{后验}} = \frac{\underbrace{P(\mathcal{D} \mid \theta)}_{\text{似然}} \cdot \underbrace{P(\theta)}_{\text{先验}}}{\underbrace{P(\mathcal{D})}_{\text{证据}}}$$

- **先验 $P(\theta)$：** 在观测数据前，对参数的信念
- **似然 $P(\mathcal{D}\mid\theta)$：** 给定参数时，数据出现的概率
- **后验 $P(\theta\mid\mathcal{D})$：** 观测数据后，对参数的更新信念
- **证据 $P(\mathcal{D})$：** 对所有可能参数积分，作为归一化常数

由于证据不依赖 $\theta$，推断中常写为：

$$P(\theta \mid \mathcal{D}) \propto P(\mathcal{D} \mid \theta) \cdot P(\theta)$$

---

## 5. 最大后验估计（Maximum A Posteriori，MAP）

### 定义

MAP 估计选择使**后验概率**最大的参数值：

$$\hat{\theta}_{\text{MAP}} = \arg\max_\theta P(\theta \mid \mathcal{D}) = \arg\max_\theta \left[\log P(\mathcal{D}\mid\theta) + \log P(\theta)\right]$$

MAP 比 MLE 多了一项 $\log P(\theta)$，这一项来自先验。

### MAP 与正则化的等价性

**L2 正则化（Ridge）与高斯先验：**

若参数先验为 $\theta \sim \mathcal{N}(0, \frac{1}{\lambda}I)$，则：

$$\log P(\theta) = -\frac{\lambda}{2}\|\theta\|_2^2 + \text{常数}$$

代入 MAP 目标：

$$\hat{\theta}_{\text{MAP}} = \arg\max_\theta \left[\log P(\mathcal{D}\mid\theta) - \frac{\lambda}{2}\|\theta\|_2^2\right]$$

即**最小化负对数似然 + L2 正则项**，这正是带 L2 正则的最大似然。

**L1 正则化（Lasso）与拉普拉斯先验：**

若先验为拉普拉斯分布（Laplace Distribution）：$P(\theta) \propto \exp(-\lambda|\theta|)$，则 MAP 等价于 L1 正则化：

$$\hat{\theta}_{\text{MAP}} = \arg\min_\theta \left[-\log P(\mathcal{D}\mid\theta) + \lambda\|\theta\|_1\right]$$

**结论：** 正则化 = 给参数加先验。正则化的强度（$\lambda$）对应先验的集中程度。这为正则化提供了贝叶斯解释：我们在说"我认为参数不应该太大"。

---

## 6. 贝叶斯推断（Bayesian Inference）

MLE 和 MAP 都只给出**点估计**（一个最优参数值）。贝叶斯推断则保留完整的**后验分布**，量化对参数的不确定性。

### 推断流程

1. **指定先验** $P(\theta)$：编码对参数的已有知识或假设
2. **写出似然** $P(\mathcal{D}\mid\theta)$：描述数据生成过程
3. **计算后验** $P(\theta\mid\mathcal{D}) \propto P(\mathcal{D}\mid\theta)\cdot P(\theta)$
4. **用后验进行预测**：

$$P(x_{\text{new}} \mid \mathcal{D}) = \int P(x_{\text{new}}\mid\theta)\cdot P(\theta\mid\mathcal{D})\,d\theta$$

这一步称为**后验预测（Posterior Predictive）**，将参数的不确定性传播到预测的不确定性。

### 共轭先验（Conjugate Prior）

若先验和后验属于同一分布族，则称先验为似然的**共轭先验**，此时后验可解析求得，无需数值积分。

| 似然 | 共轭先验 | 后验 |
|------|----------|------|
| 伯努利/二项 | Beta | Beta |
| 类别/多项 | Dirichlet | Dirichlet |
| 高斯（均值未知） | 高斯 | 高斯 |
| 泊松 | Gamma | Gamma |

### 贝叶斯更新的直觉

贝叶斯推断的本质是**渐进学习**：

$$\underbrace{P(\theta\mid\mathcal{D}_1, \mathcal{D}_2)}_{\text{两批数据后验}} \propto P(\mathcal{D}_2\mid\theta)\cdot\underbrace{P(\theta\mid\mathcal{D}_1)}_{\text{第一批数据后验作为先验}}$$

每次看到新数据，后验更新为新的先验，继续迭代。数据越多，后验越集中（不确定性降低）。

---

## 7. MLE、MAP、贝叶斯推断的对比

| 方法 | 输出 | 是否用先验 | 是否量化不确定性 | 计算复杂度 |
|------|------|-----------|-----------------|------------|
| MLE | 点估计 $\hat\theta$ | 否 | 否 | 低 |
| MAP | 点估计 $\hat\theta$ | 是 | 否 | 低 |
| 贝叶斯推断 | 后验分布 $P(\theta\mid\mathcal{D})$ | 是 | 是 | 高（通常需要近似） |

**实践中的选择：**

- 大数据场景（深度学习）：MLE 为主，L2/L1 正则视为 MAP 的近似
- 小数据场景：MAP 或完整贝叶斯推断更稳健
- 需要不确定性估计（医疗、自动驾驶）：贝叶斯方法或不确定性量化（如 MC Dropout）

---

## 8. 变分推断简介（Variational Inference）

完整的贝叶斯推断往往因积分难以解析计算。**变分推断（Variational Inference，VI）** 是一种近似方法：

用一个参数化的简单分布 $q_\phi(\theta)$ 来近似后验 $P(\theta\mid\mathcal{D})$，通过优化使二者尽量接近（最小化 KL 散度）。

$$\phi^* = \arg\min_\phi \text{KL}(q_\phi(\theta) \| P(\theta\mid\mathcal{D}))$$

变分自编码器（VAE）正是将变分推断与神经网络结合的典范。

---

## 9. 本节小结

```
数据 D + 模型假设（似然）
         ↓
最大似然估计 (MLE)          → 点估计，无先验
         +
先验 P(θ)
         ↓
最大后验估计 (MAP)          → 点估计，含先验（等价于正则化）
         ↓
贝叶斯推断                  → 完整后验，量化不确定性
```

**三句话记住核心联系：**

1. 交叉熵损失 = 类别分布假设下的 MLE
2. 均方误差损失 = 高斯噪声假设下的 MLE
3. L2/L1 正则化 = 高斯/拉普拉斯先验下的 MAP
