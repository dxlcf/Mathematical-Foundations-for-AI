# 概率论与数理统计（Probability Theory & Mathematical Statistics）概述

## 为什么 AI 需要概率论？

AI 系统面对的世界本质上是不确定的：数据有噪声，标注可能有误，未来的输入无法预知。

概率论提供了一套语言，让我们可以**精确地描述不确定性**，而不是回避它。

机器学习的核心问题，几乎都可以用概率的视角重新表述：

| 任务 | 概率视角 |
|------|----------|
| 分类 | 估计条件概率 $P(y \mid x)$ |
| 生成模型 | 学习数据分布 $P(x)$ |
| 贝叶斯推断 | 用先验 + 数据更新信念 |
| 损失函数设计 | 最大化似然或后验 |
| 正则化 | 施加参数的先验分布 |

## 学习建议

**优先掌握的核心能力：**

1. 能写出并理解条件概率和贝叶斯公式
2. 能识别高斯分布，理解均值与方差的含义
3. 能推导简单模型的最大似然估计
4. 能理解协方差矩阵的几何含义

**可暂缓深入的内容：**

- 测度论层面的严格概率定义
- 复杂分布的矩母函数推导
- 非参数统计方法
